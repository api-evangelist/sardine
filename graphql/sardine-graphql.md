# Sardine GraphQL API

Sardine provides fraud prevention and compliance for fintech and crypto. The API covers device intelligence, behavioral biometrics, transaction risk scoring, AML screening, KYC verification, and chargeback dispute management.

**Endpoint:** No native GraphQL endpoint — conceptual schema derived from Sardine REST API
**Documentation:** https://docs.sardine.ai/home
