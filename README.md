# Sardine

Sardine is an agentic financial crime platform offering fraud prevention, AML compliance, and transaction monitoring through a unified REST API. The platform provides real-time device intelligence, behavior biometrics, transaction risk scoring, KYC/KYB identity verification, AML screening, and chargeback protection for banks, fintechs, and commerce at enterprise scale. Sardine serves 450+ enterprise customers and has screened over $1.48 trillion in payments.

## APIs

- **Sardine API** - Unified REST API covering Device Intelligence, Customer Risk (KYC/KYB), Crypto On/Off-Ramp, Card Issuing, and Transaction Monitoring (AML/Sanctions)
